**BW Nexus AI** is a next-generation multi-agent intelligence platform designed to identify hidden opportunities, predict strategic outcomes, and design high-impact pathways for governments, investors, and multilateral institutions.

Powered by **Google Gemini 2.5 & 3.0 Pro** models, it moves beyond traditional analytics by integrating economics, governance, culture, and sentiment into a unified intelligence model.

## üöÄ Core Engines

### 1. Nexus Rocket Engine
The flagship capabilities for high-velocity economic analysis:
*   **Latent Asset Identification (LAI):** Scans regions for underutilized assets and cross-domain potentials (e.g., skills + geography + infrastructure).
*   **Investment Velocity & Activation Score (IVAS):** Calculates how fast an opportunity can be activated based on regulatory friction and political will.
*   **Symbiotic Cascade Forecast (SCF):** Models second-order economic effects, including job creation and fiscal impact.

### 2. Strategic Canvas
A modular workspace that adapts to the user's mission (e.g., "Attract Investment", "Enter Market"). Modules include:
*   **Geopolitical & Economic Forecast:** Real-time stability and risk assessment.
*   **RROI Diagnostic:** Regional Readiness & Opportunity Index.
*   **Governance Integrity Audit:** Compliance and corruption risk analysis.
*   **Symbiotic Matchmaking:** Finds partners with high asymmetry leverage.

## üì¶ Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-org/bw-nexus-ai.git
    cd bw-nexus-ai
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Configure Environment**
    Create a `.env` file in the root directory and add your Google Gemini API Key:
    ```env
    API_KEY=your_google_gemini_api_key_here
    ```
    *Note: Access to `gemini-3-pro-preview` and `gemini-2.5-flash` models is required.*

4.  **Start the Application**
    ```bash
    npm start
    ```

## üß≠ Usage Guide

1.  **Identity & Context:** Launch the **Intelligence Design Studio**. Define your organization type (e.g., Government, MNC) and target region.
2.  **Select Intent:** Choose a strategic mission (e.g., "Attract Foreign Investment"). The system will recommend specific intelligence modules.
3.  **Strategic Canvas:** Enter the interactive workspace. Activate modules like the **Rocket Engine** to generate synthetic data and run forecasts.
4.  **Live Intelligence:** Use the **Command Center** to access real-time global feeds and agent diagnostics.
5.  **Export:** Generate a comprehensive **NSIL Dossier** containing all analysis, charts, and strategic roadmaps.
